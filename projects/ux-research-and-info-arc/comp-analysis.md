# Competitive Analysys Report

Findings from a comparative review of six developer platforms (npm, Artifact Hub, Cratesio, Docker
Hub, Go Package site and the OpenTelemetry Ecosystem Explorer) to understand how developer tools
organize, present and maintain package and component information.
